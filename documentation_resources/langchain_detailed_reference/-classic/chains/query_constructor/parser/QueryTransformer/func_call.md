# func\_call

Transform a function name and args into a FilterDirective.


```
func_call(
    self,
    func_name: Any,
    args: list,
) -> FilterDirective
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `func_name`\* | `Any` | The name of the function. |
| `args`\* | `list` | The arguments passed to the function. |
