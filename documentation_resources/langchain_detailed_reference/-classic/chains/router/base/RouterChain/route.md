# route

Route inputs to a destination chain.


```
route(
  self,
  inputs: dict[str, Any],
  callbacks: Callbacks = None
) -> Route
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `inputs`\* | `dict[str, Any]` | inputs to the chain |
| `callbacks` | `Callbacks` | Default:`None`  callbacks to use for the chain |
