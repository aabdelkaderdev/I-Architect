# go\_to\_page

Navigate to the given URL.


```
go_to_page(
    self,
    url: str,
) -> None
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `url`\* | `str` | The URL to navigate to. If it does not contain a scheme, it will be prefixed with "http://". |
