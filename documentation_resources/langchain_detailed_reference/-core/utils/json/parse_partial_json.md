# parse\_partial\_json

Parse a JSON string that may be missing closing braces.


```
parse_partial_json(
    s: str,
    *,
    strict: bool = False,
) -> Any
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `s`\* | `str` | The JSON string to parse. |
| `strict` | `bool` | Default:`False`  Whether to use strict parsing. |
