# xor\_args

Validate specified keyword args are mutually exclusive.


```
xor_args(
    *arg_groups: tuple[str, ...] = (),
) -> Callable
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `*arg_groups` | `tuple[str, ...]` | Default:`()`  Groups of mutually exclusive keyword args. |
