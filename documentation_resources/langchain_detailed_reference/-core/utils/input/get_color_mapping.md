# get\_color\_mapping

Get mapping for items to a support color.


```
get_color_mapping(
  items: list[str],
  excluded_colors: list | None = None
) -> dict[str, str]
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `items`\* | `list[str]` | The items to map to colors. |
| `excluded_colors` | `list | None` | Default:`None`  The colors to exclude. |
