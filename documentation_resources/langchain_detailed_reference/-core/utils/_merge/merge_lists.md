# merge\_lists

Add many lists, handling `None`.


```
merge_lists(
    left: list | None,
    *others: list | None = (),
) -> list | None
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `left`\* | `list | None` | The first list to merge. |
| `others` | `list | None` | Default:`()`  The other lists to merge. |
