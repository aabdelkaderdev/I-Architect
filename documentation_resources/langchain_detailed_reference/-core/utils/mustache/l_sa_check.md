# l\_sa\_check

Do a preliminary check to see if a tag could be a standalone.


```
l_sa_check(
  template: str,
  literal: str,
  is_standalone: bool
) -> bool
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `template`\* | `str` | The template. (Not used.) |
| `literal`\* | `str` | The literal. |
| `is_standalone`\* | `bool` | Whether the tag is standalone. |
