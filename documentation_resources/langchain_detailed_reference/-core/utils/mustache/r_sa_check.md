# r\_sa\_check

Do a final check to see if a tag could be a standalone.


```
r_sa_check(
  template: str,
  tag_type: str,
  is_standalone: bool
) -> bool
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `template`\* | `str` | The template. |
| `tag_type`\* | `str` | The type of the tag. |
| `is_standalone`\* | `bool` | Whether the tag is standalone. |
