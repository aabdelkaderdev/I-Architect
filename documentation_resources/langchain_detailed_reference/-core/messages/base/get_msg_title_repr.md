# get\_msg\_title\_repr

```
    bold: bool = False,
) -> str
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `title`\* | `str` | The title. |
| `bold` | `bool` | Default:`False`  Whether to bold the title. |
