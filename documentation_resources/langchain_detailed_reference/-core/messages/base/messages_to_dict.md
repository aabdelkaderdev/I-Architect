# messages\_to\_dict

Convert a sequence of Messages to a list of dictionaries.


```
messages_to_dict(
    messages: Sequence[BaseMessage],
) -> list[dict]
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `messages`\* | `Sequence[BaseMessage]` | Sequence of messages (as `BaseMessage`s) to convert. |
