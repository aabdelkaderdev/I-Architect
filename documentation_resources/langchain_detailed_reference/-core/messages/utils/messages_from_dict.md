# messages\_from\_dict

Convert a sequence of messages from dicts to `Message` objects.


```
messages_from_dict(
    messages: Sequence[dict],
) -> list[BaseMessage]
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `messages`\* | `Sequence[dict]` | Sequence of messages (as dicts) to convert. |
