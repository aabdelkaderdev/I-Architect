# VideoContentBlock

Video data.

Factory function

`create_video_block` may also be used as a factory to create a
`VideoContentBlock`. Benefits include:

- Automatic ID generation (when not provided)
- Required arguments strictly validated at creation time


```
VideoContentBlock()
```
