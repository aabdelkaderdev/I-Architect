# AudioContentBlock

Audio data.

Factory function

`create_audio_block` may also be used as a factory to create an
`AudioContentBlock`. Benefits include:

- Automatic ID generation (when not provided)
- Required arguments strictly validated at creation time


```
AudioContentBlock()
```
