# sorted\_values

Return a list of values in dict sorted by key.


```
sorted_values(
    values: dict[str, str],
) -> list[Any]
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `values`\* | `dict[str, str]` | A dictionary with keys as input variables and values as their values. |
