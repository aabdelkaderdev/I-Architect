# merge\_chat\_generation\_chunks

```
    chunks: list[ChatGenerationChunk],
) -> ChatGenerationChunk | None
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `chunks`\* | `list[ChatGenerationChunk]` | A list of `ChatGenerationChunk` to merge. |
