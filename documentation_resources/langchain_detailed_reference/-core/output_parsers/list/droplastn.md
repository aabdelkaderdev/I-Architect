# droplastn

Drop the last `n` elements of an iterator.


```
droplastn(
    iter: Iterator[T],
    n: int,
) -> Iterator[T]
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `iter`\* | `Iterator[T]` | The iterator to drop elements from. |
| `n`\* | `int` | The number of elements to drop. |
