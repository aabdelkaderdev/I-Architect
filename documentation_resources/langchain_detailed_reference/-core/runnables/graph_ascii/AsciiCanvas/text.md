# text

Print a text on ASCII canvas.


```
text(
    self,
    x: int,
    y: int,
    text: str,
) -> None
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `x`\* | `int` | x coordinate where the text should start. |
| `y`\* | `int` | y coordinate where the text should start. |
| `text`\* | `str` | string that should be printed. |
