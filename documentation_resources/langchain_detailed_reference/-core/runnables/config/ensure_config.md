# ensure\_config

Ensure that a config is a dict with all keys present.


```
ensure_config(
    config: RunnableConfig | None = None,
) -> RunnableConfig
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `config` | `RunnableConfig | None` | Default:`None`  The config to ensure. |
