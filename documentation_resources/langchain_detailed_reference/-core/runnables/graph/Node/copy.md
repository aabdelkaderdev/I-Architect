# copy

Return a copy of the node with optional new id and name.


```
copy(
  self,
  *,
  id: str | None = None,
  name: str | None = None
) -> Node
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `id` | `str | None` | Default:`None`  The new node id. |
| `name` | `str | None` | Default:`None`  The new node name. |
