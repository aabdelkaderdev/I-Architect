# add\_subgraph

Add subgraphs to the graph.


```
add_subgraph(
  self,
  viz: Any,
  nodes: list[list[str]],
  parent_prefix: list[str] | None = None
) -> None
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `viz`\* | `Any` | The graphviz object. |
| `nodes`\* | `list[list[str]]` | The nodes to add. |
| `parent_prefix` | `list[str] | None` | Default:`None`  The prefix of the parent subgraph. |
