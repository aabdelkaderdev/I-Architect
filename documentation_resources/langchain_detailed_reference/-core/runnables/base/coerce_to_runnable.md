# coerce\_to\_runnable

Coerce a `Runnable`-like object into a `Runnable`.


```
coerce_to_runnable(
    thing: RunnableLike,
) -> Runnable[Input, Output]
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `thing`\* | `RunnableLike` | A `Runnable`-like object. |
