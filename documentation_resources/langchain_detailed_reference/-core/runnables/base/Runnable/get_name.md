# get\_name

Get the name of the `Runnable`.


```
get_name(
  self,
  suffix: str | None = None,
  *,
  name: str | None = None
) -> str
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `suffix` | `str | None` | Default:`None`  An optional suffix to append to the name. |
| `name` | `str | None` | Default:`None`  An optional name to use instead of the `Runnable`'s name. |
