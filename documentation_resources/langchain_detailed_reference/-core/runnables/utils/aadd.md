# aadd

Asynchronously add a sequence of addable objects together.


```
aadd(
    addables: AsyncIterable[Addable],
) -> Addable | None
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `addables`\* | `AsyncIterable[Addable]` | The addable objects to add. |
