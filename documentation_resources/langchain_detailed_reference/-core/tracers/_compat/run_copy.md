# run\_copy

Copy run, compatible with both Pydantic v1 and v2.


```

    run: Run,
    **kwargs: Any = {},
) -> Run
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `run`\* | `Run` | The run to copy. |
| `**kwargs` | `Any` | Default:`{}`  Additional arguments passed to `model_copy`/`copy`. |
