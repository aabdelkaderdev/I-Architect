# try\_json\_stringify

Try to stringify an object to JSON.


```
try_json_stringify(
    obj: Any,
    fallback: str,
) -> str
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `obj`\* | `Any` | Object to stringify. |
| `fallback`\* | `str` | Fallback string to return if the object cannot be stringified. |
