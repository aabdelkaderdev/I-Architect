# elapsed

Get the elapsed time of a run.


```
elapsed(
    run: Any,
) -> str
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `run`\* | `Any` | any object with a `start_time` and `end_time` attribute. |
