# add\_tags

Add tags to the callback manager.


```
add_tags(
    self,
    tags: list[str],
    inherit: bool = True,
) -> None
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `tags`\* | `list[str]` | The tags to add. |
| `inherit` | `bool` | Default:`True`  Whether to inherit the tags. |
