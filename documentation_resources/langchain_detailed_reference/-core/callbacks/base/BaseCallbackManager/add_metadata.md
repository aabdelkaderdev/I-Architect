# add\_metadata

Add metadata to the callback manager.


```
add_metadata(
  self,
  metadata: dict[str, Any],
  inherit: bool = True
) -> None
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `metadata`\* | `dict[str, Any]` | The metadata to add. |
| `inherit` | `bool` | Default:`True`  Whether to inherit the metadata. |
