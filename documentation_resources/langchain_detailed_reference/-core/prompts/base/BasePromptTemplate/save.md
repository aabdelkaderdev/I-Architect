# save

Save the prompt.


```
save(
    self,
    file_path: Path | str,
) -> None
```

**Example:**

```
prompt.save(file_path="path/prompt.yaml")
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `file_path`\* | `Path | str` | Path to directory to save prompt to. |
