# get\_lc\_namespace

Get the namespace of the LangChain object.

For example, if the class is `langchain.llms.openai.OpenAI`, then the namespace
is `["langchain", "llms", "openai"]`


```
get_lc_namespace(
    cls,
) -> list[str]
```
