# format

Format the prompt with the inputs.


```
format(
    self,
    **kwargs: Any = {},
) -> str
```

**Example:**

```
prompt.format(variable1="foo")
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `**kwargs` | `Any` | Default:`{}`  Any arguments to be passed to the prompt template. |
