# tool\_invoke\_params\_example

Returns a dictionary representing the "args" of an example tool call.

This should NOT be a `ToolCall` dict - it should not have
`{"name", "id", "args"}` keys.


```
tool_invoke_params_example: dict[str, Any]
```
