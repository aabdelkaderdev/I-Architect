# unicode\_customer

Tool for creating a customer with Unicode name.


```
unicode_customer(
    customer_name: str,
    description: str,
) -> str
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `customer_name`\* | `str` | The customer's name in their native language. |
| `description`\* | `str` | Description of the customer. |
