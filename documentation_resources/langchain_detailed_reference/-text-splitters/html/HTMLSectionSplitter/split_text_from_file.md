# split\_text\_from\_file

Split HTML content from a file into a list of `Document` objects.


```
split_text_from_file(
    self,
    file: StringIO,
) -> list[Document]
```

## Parameters

| Name | Type | Description |
| --- | --- | --- |
| `file`\* | `StringIO` | A file path or a file-like object containing HTML content. |
