# LangChain reference

Welcome to the [LangChain](https://github.com/langchain-ai/langchain) package reference documentation!

Most users will primarily interact with the main [`langchain`](/python/langchain/) package, which provides the complete set of implementations for building LLM applications. The packages below form the foundation of the LangChain ecosystem, each serving a specific purpose in the architecture:

[`langchain`

The main entrypoint containing all implementations you need for building applications with LLMs.](/python/langchain/)
[`langchain-core`

Core interfaces and abstractions used across the LangChain ecosystem.](/python/langchain-core/)
[`langchain-text-splitters`

Text splitting utilities for document processing.](/python/langchain-text-splitters/)
[`langchain-mcp-adapters`

Make MCP tools available in LangChain and LangGraph applications.](/python/langchain-mcp-adapters/)
[`langchain-tests`

Standard tests suite used to validate LangChain integration package implementations.](/python/langchain-tests/)
[`langchain-classic`

Legacy `langchain` implementations and components.](/python/langchain-classic/)

Integration Packages
