# Resources

## **PyMuPDF Pro**

For **Office** file support [try PyMuPDF Pro](pymupdf-pro/index.html#pymupdf-pro).

---

## Find out about **PyMuPDF Utilities**

The GitHub repository [PyMuPDF-Utilities](https://github.com/pymupdf/PyMuPDF-Utilities) contains a full range of examples, demonstrations and use cases.

---

## Do you need PDF to **DOCX** conversion?

We recommend the [pdf2docx](https://pdf2docx.readthedocs.io/en/latest/) library which uses PyMuPDF and the **python-docx** library to provide simple document conversion from PDF to **DOCX** format.
