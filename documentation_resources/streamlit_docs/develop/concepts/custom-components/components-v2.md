Components v2 represents a reimagining of how custom components work in Streamlit. It's designed to unlock new capabilities and dramatically simplify the development experience. To view the command reference, see the [API Reference](/develop/api-reference/custom-components).

Create instances of your component in your app and handle their output.](/develop/concepts/custom-components/components-v2/mount)[#### Bidirectional communication

Exchange data between your component and Python.](/develop/concepts/custom-components/components-v2/communicate)[#### State vs triggers

Understand the two communication mechanisms for building interactive components.](/develop/concepts/custom-components/components-v2/state-and-triggers)[#### Theming and styling

Make your components look great with Streamlit's theme integration and CSS custom properties.](/develop/concepts/custom-components/components-v2/theming)[#### Package-based components

Build distributable components with TypeScript, external dependencies, and the official component template.](/develop/concepts/custom-components/components-v2/package-based)

- [streamlit-bokeh v2 migration](https://github.com/streamlit/streamlit-bokeh/pull/40)
