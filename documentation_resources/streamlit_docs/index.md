#### Dynamic containers

`st.tabs`, `st.expander`, and `st.popover` can track open/closed state and trigger reruns with the new `on_change` parameter.](/develop/api-reference/layout/st.tabs?utm_source=streamlit)[*share*

#### Widget binding

Most non-trigger widgets have a `bind` parameter to sync widget state with URL query parameters.](/develop/concepts/architecture/widget-behavior?utm_source=streamlit#binding-widgets-to-query-parameters)[*open\_in\_new*

#### Clickable images

`st.image` has a `link` parameter to make images clickable with HTTP/HTTPS URLs.](/develop/api-reference/media/st.image?utm_source=streamlit)[*visibility\_off*

#### Hidden pages

`st.Page` has a `visibility` parameter to hide pages in the navigation menu while keeping them routable.](/develop/api-reference/navigation/st.page?utm_source=streamlit)[*format\_color\_text*

#### CSS colors in Markdown

Markdown supports arbitrary CSS colors for text foreground and background styling.](/develop/api-reference/text/st.markdown?utm_source=streamlit)[*speed*

#### Metric delta descriptions

`st.metric` has a `delta_description` parameter to display descriptive text next to delta values.](/develop/api-reference/data/st.metric?utm_source=streamlit)
